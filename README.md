# 🌍 React Translator App

A modern translator web application built using React and Tailwind CSS.

---

## 🚀 Features

- 🌐 Translate text into multiple languages  
- 🎤 Speech-to-text input  
- 🔊 Text-to-speech output  
- 📋 Copy & Paste functionality  
- ❌ Clear input instantly  
- 🔐 Login & Signup UI  

---

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS
- Axios
- Translation API
- Web Speech API
